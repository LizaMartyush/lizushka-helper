#  Шпаргалка по практикуму
##  Шпаргалка для Лизушки
git log показывает хэш, автора и дату коммита
git log --oneline показывает лог одной строкой
Последовательность для обновление информации внутри файлов:
1) Обновить текстовый файл
2) git status получить статус файла
3) git add наименование файла, например, README.md добавить обновление
4) git commit -m'наименование коммита' закоммитить обновление
5) git push запушить в гитхаб
6) git log получить логи по коммитам
